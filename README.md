# Crescendo-2024
2024 Robot Code

If you are deploying code to the robot at a competition PLEASE make sure you have followed [these steps](https://github.com/Mechanical-Advantage/AdvantageKit/blob/main/docs/VERSION-CONTROL.md#event-deploy)


Make sure AdvantageKit is updated with each WPILib update otherwise stuff will break.<br>
Also make sure the `org.junit.platform:junit-platform-launcher` in build.gradle has the appropriate *latest* version concatenated at the end. (ex. `org.junit.platform:junit-platform-launcher:1.10.2`)
