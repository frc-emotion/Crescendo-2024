If you are deploying code to the robot at a competition PLEASE make sure you have followed (https://github.com/Mechanical-Advantage/AdvantageKit/blob/main/docs/VERSION-CONTROL.md#event-deploy)[these steps]

Make sure AdvantageKit is updated with each WPILib update otherwise stuff will break.
Also make sure the `org.junit.platform:junit-platform-launcher` in build.gradle has the appropriate *latest* version concatenated at the end. (ex. ':1.10.2')
